# Ubuntu 丢失网卡

终端运行：

```shell
sudo service network-manager restart
```

请注意，在某些发行版中（例如较新的Ubuntu版本），推荐使用 `systemctl` 来管理服务，因此对应的命令应该是：

```shell
sudo systemctl restart NetworkManager
```
